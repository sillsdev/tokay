Chorus Hub is a simple way to Send/Receive on a local network, like an office or home. Better instructions will follow in a future version, but for now, this should get you started: First designate one computer on your network as "the server", and make sure this product is installed on it. Next, look in the directory where this product is installed, and run the program labeled "ChorusHub.exe". If all goes well, there's nothing more to do! Now from any other computer on the network, you can Click "ChorusHub" to Send/Receive to it.
ChorusHub does everything it can to be "Plug n Play". However, some things that may need your attention. 1) ChorusHub will attempt to create a folder at "C:\ChorusHub". It may need administrator-levl permissions to do that. Any technical support person can help you run ChorusHub.exe with the necessary permissions. 2) Your firewall may need to give permission for the server and the client machines to talk to each other on this 'port'.
